## Условие задачи (Занятие 9)

Определить структурный тип, описывающий маркетплейсы (название
маркетплейса, количество ПВЗ по стране, город дислокации, рыночная стоимость,
тип маркетплейса: продуктовый, товары для дома, все-подрял; тип бизнеса
компании: основной, вспомогательный, франшиза и др.). Заполнить структурный
массив 20-ю записями. Переписать из исходного массива в другой массив,
информацию только о маркетплейсах с товарами для дома. Затем новый массив
отсортировать по рыночной стоимости. Вывести все данные по конкретному
маркетплейсу. Вывести 2 маркетплейса из одного города, но не из Москвы.
Реализовать функцию изменения данных о маркетплейсе. В отдельный массив
поместить все маркетплейсы Москвы. Реализовать вывод отфильтрованных данных
в виде оберточной функции.


Задачи:
1. Переписать из исходного массива в другой массив информацию только о маркетплейсах с товарами для дома. Затем новый массив отсортировать по рыночной стоимости.
2. Вывести все данные по конкретному маркетплейсу.
3. Вывести 2 маркетплейса из одного города, но не из Москвы.
4. Реализовать функцию изменения данных о маркетплейсе.
5. В отдельный массив поместить все маркетплейсы из Москвы.
6. Реализовать вывод отфильтрованных данных в виде оберточной функции.



## Условие задачи (Занятие 10)

Задание 1 (текстовый файл):
Считать из текстового файла данные числового поля (количество ПВЗ). Если название маркетплейса совпадает с элементом массива структур, то в поле pvz_count записывается соответствующее значение из файла.

Задание 2 (бинарный файл):
Сделать чтение и запись всего массива структуры Marketplace в бинарный файл. Оформить в виде отдельных подпрограмм.
